# Blinkit Sales Analysis Dashboard

## 📊 Overview
The Blinkit Sales Analysis Dashboard is a Power BI project that visualizes and analyzes sales data from Blinkit, India's last-minute grocery delivery platform.

This dashboard provides insights into sales trends, outlet performance, item categories, and customer preferences.

---

## 🚀 Key Metrics

- **Total Sales:** $1.19M  
- **Average Rating:** 3.9  
- **Average Sales per Item:** $141  
- **Total Number of Items:** 8406  

---

## 📌 Dashboard Features

### 1. Sales Overview
- Displays total sales, average sales, and number of items
- Helps understand overall business performance

### 2. Outlet Establishment Trend
- Year-wise growth of outlets
- Identifies expansion patterns over time

### 3. Outlet Analysis
- Sales by outlet size (Small, Medium, High)
- Sales by outlet location (Tier 1, Tier 2, Tier 3)
- Sales by outlet type (Supermarket, Grocery Store)

### 4. Product Insights
- Sales distribution by item type (Fruits, Snacks, Dairy, etc.)
- Helps identify top-performing categories

### 5. Fat Content Analysis
- Sales comparison between Low Fat and Regular items
- Outlet-wise fat content distribution

### 6. Customer Insights
- Average ratings across outlet types
- Item visibility impact on sales

---

## 🎛️ Filters Available

Users can interact with the dashboard using:
- Outlet Location Type
- Outlet Size
- Item Type

---

## 🛠️ Tools & Technologies Used

- **Power BI** – Data Visualization
- **Excel** – Data Source
- **Data Cleaning & Transformation**
- **DAX (Data Analysis Expressions)**

---

## 💡 Key Insights

- Tier 3 outlets generate the highest sales.
- Medium-sized outlets contribute the most revenue.
- Fruits and Snacks are the top-selling categories.
- Regular fat items dominate sales compared to low-fat items.
- Sales peaked around 2018 due to outlet expansion.
